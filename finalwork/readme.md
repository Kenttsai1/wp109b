# 來源
本程式修改自https://www.youtube.com/watch?v=bG2BmmYr9NQ&t=321s&ab_channel=KnifeCircusKnifeCircus

# 功能
```
1.利用滑鼠點擊使pikachu做出跳躍的動作
2.跳躍有製造出一點懸停的感覺
3.使用CSS中的動畫
4.計分
5.背景

```

# 技術手段
```
 新增跳躍的角色圖片（可愛的皮卡丘）
 弄成gif感覺更有實感
 背景調整
 大小調整使玩家的眼睛不那麼疲勞
 寫上註解
 增加遊戲難度 如分數越高速度越快
 有一個音樂播放器
```
